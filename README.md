# Credit Risk Prediction 💳

This project focuses on building a machine learning model to predict the credit risk of loan applicants. I used a Jupyter Notebook to analyze the data, visualize features, and train the model.

## 🚀 Key Achievements
* **High Accuracy:** The final model achieved a predictive accuracy of **93.9%**.
* **Data Visualization:** Included detailed charts for Exploratory Data Analysis (EDA).

## 🛠️ Technologies Used
* **Language:** Python
* **Platform:** Jupyter Notebook
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn

## 📂 Project Structure
* `CreditRisk.ipynb`: The main notebook containing all the Python code, data analysis, and model training.
